来自于一位受够了网上富文本编辑器过于囊肿难以扩展的朋友开发了这款编辑器！

  • 依赖jQuery
  • 依赖HTML5 contenteditable

引用

<link rel="stylesheet" type="text/css" href="editor.css">
<link rel="stylesheet" type="text/css" href="icon/iconfont.css">
<script type="text/javascript" src="jquery.min.js"></script>
<script type="text/javascript" src="editor.js"></script>

使用

<div class="hy-editor-container" contenteditable="true">
放个初始内容
</div>
<script type="text/javascript">
	//初始编辑器方法1
	var test = new HY_editor('.hy-editor',{
		toolbar:'image justifyleft justifycenter justifyright | bold italic | underline strike blockquote orderedList unorderedList hr undo redo'
	});
	//初始编辑器方法2
	var test1 = new HY_editor('#hy-editor',{
		toolbar:'italic bold  underline strike blockquote orderedList unorderedList hr undo redo'
	}); 
</script>
